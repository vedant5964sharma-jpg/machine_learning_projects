# California Housing Regression Project

This project implements Linear Regression on the California Housing dataset.

## Steps Implemented

1. Data Loading
2. Exploratory Data Analysis (EDA)
3. Feature correlation analysis
4. Train-test split
5. Linear Regression model training
6. Model evaluation using RMSE, MAE and R2 score
7. Cross validation

## Dataset

California Housing Dataset

Target variable:
MedHouseVal

## Tools Used

Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
